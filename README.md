# 名古屋市立大学 河合勝彦研究室 (KKLab) ウェブサイト

名古屋市立大学 河合勝彦研究室（KKLab）の公式ウェブサイトのソースです。
「経営とITの融合」をテーマに、情報技術・経済学・経営学を統合した学際的研究に取り組んでいます。

- 公開URL: <https://kklab.mobi/>
- GitHub（研究室）: <https://github.com/orgs/kkawailab/repositories>

## サイト構成

3つのページで構成しています。

| ページ | ファイル | 内容 |
| --- | --- | --- |
| 研究室紹介 | `index.html` | 研究室の理念・研究テーマ・研究手法、所属学生へのメッセージ、関連リンク |
| メンバー紹介 | `members.html` | 教員（河合勝彦 教授）および所属学生（学年別） |
| 研究成果 | `works.html` | 公開しているアプリ・サイト（注目の成果＋カテゴリ別一覧） |

## ディレクトリ構成

```
.
├── index.html          研究室紹介（トップページ）
├── members.html        メンバー紹介
├── works.html          研究成果
├── assets/
│   └── style.css       全ページ共通のスタイルシート
├── imgs/               画像
│   └── mountain.png
├── _config.yml         Jekyll 設定（タイトル・説明など）
├── CNAME               カスタムドメイン設定（kklab.mobi）
└── .github/workflows/
    └── jekyll-gh-pages.yml   GitHub Pages への自動デプロイ
```

## 技術構成

- **プレーン HTML + 共通 CSS**（ビルドツール・JS フレームワークなし）
- スタイルは `assets/style.css` に一元化。配色などは先頭の `:root` に CSS 変数（デザイントークン）としてまとめています。
- フォント: Noto Serif JP（見出し）/ Noto Sans JP（本文）
- ホスティング: **GitHub Pages**（独自ドメイン `kklab.mobi`、`main` ブランチへの push で自動デプロイ）

## ローカルでの確認方法

CSS をルート絶対パス（`/assets/style.css`）で読み込んでいるため、`file://` で直接開くとスタイルが適用されません。
ローカル HTTP サーバー経由で確認してください。

```bash
# リポジトリ直下で実行
python3 -m http.server 8000
# ブラウザで http://localhost:8000/ を開く
```

## 編集・更新方法

- **デザインの調整**: `assets/style.css` 冒頭の `:root` にある CSS 変数（色・余白・フォントサイズ）を編集します。
- **メンバーの追加**: `members.html` の各学年セクションにある `<li class="placeholder">` を実データに置き換えます。手順は同ファイル内の HTML コメントに記載しています。
- **研究成果の追加**: `works.html` の「注目の成果」は `<article class="card feature">`、「カテゴリ別一覧」は `<li class="item">` をコピーして編集します。
- **共通部分（ヘッダー/フッター）**: テンプレート機構がないため、変更時は 3 ページすべてに同じ修正を反映してください。

## デプロイ

`main` ブランチへ push すると、GitHub Actions（"Deploy Jekyll site to Pages"）が実行され、数分で <https://kklab.mobi/> に反映されます。
