研究成果

論文だけでなく、研究や教育の成果を「動くアプリ・サイト」として公開しています。ブラウザですぐに試せるシミュレーションやデータ可視化、学習用の教材サイトなどを取り揃えています。

注目の成果

資産格差シミュレーション

p5.jsエージェントベースモデル

ランダムな経済取引から格差がどのように生まれるか（Yard-Sale モデル）を、エージェントベースで可視化するシミュレーションです。

デモを見る GitHub

名古屋市コンビニ分布の地図分析

Leaflet地理空間分析

名古屋市内のコンビニエンスストアの分布を可視化・分析するインタラクティブな地図アプリ。ヒートマップやクラスター分析に対応しています。

デモを見る GitHub

e-Stat 統計データ可視化アプリ

JavaScriptデータ可視化

日本政府の統計データ（e-Stat API）を、モダンな UI でインタラクティブに可視化するクライアントサイドアプリです。
